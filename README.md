# Chord Compass

A music academy management system for teaching Western Piano, Western Music Theory, Carnatic Vocals and Carnatic Keyboard.

## Features (Planned)
- Student portfolio and course catalog
- Online course enrollment
- Class schedule calendar with Google Meet integration
- Fee payment tracking
- Course materials access for onboarded students
- Admin console for managing students and courses

## Tech Stack
- Backend: Java + Spring Boot
- Frontend: React
- Database: PostgreSQL
- DevOps: Docker, Kubernetes
- Cloud: AWS/Azure/GCP
